# minni-siem
This is a python-based mini siem that looks for system heuristics and reports findings of MITRE ATT&amp;CK issues
Using terminal, navigate to the source folder, run the script using [python siem.py log/] without the brackets. 
The script shoudl aprse the log files from the local computer, convert to JSON and then compare against open MITRE known vulnerabilities.
